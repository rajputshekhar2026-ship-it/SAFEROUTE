## 📖 Project Overview

Safe-Route Navigation System is a comprehensive safety solution for women and vulnerable users, providing intelligent route planning with real-time risk assessment, emergency SOS, and discreet safety features.

### Key Features
- 🗺️ **Smart Route Planning** - Fastest, safest, and well-lit routes
- 🚨 **Real-time SOS** - Instant emergency alerts with location tracking
- 📍 **Live Location Tracking** - WebSocket-based real-time monitoring
- 🎯 **Crime Prediction** - ML-based risk scoring (0-100)
- 🏥 **Safe Refuge Locator** - Nearby police, hospitals, and safe spaces
- 🎭 **Health Mode** - UI disguise for discreet usage
- ⌚ **Smart Watch Integration** - Apple Watch & Wear OS support
- 📸 **Incident Reporting** - Report with photo/audio evidence
