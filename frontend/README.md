# Safe-Route Navigation System - Mobile App

A production-ready React Native mobile application for personal safety during navigation, featuring real-time risk assessment, SOS functionality, and disguise mode.

## Features

- 🗺️ Real-time map with multiple route options (fastest, safest, lit streets)
- 🚨 One-tap SOS with location, audio, and photo capture
- 🔄 Real-time safety monitoring with deviation detection
- 🎭 Health mode (UI disguise as weather/news app)
- 📞 Fake call feature with conversation prompts
- ⌚ Apple Watch integration for route preview and alerts
- 📍 Offline mode with cached safe zones
- 🎤 Voice guidance for turn-by-turn navigation
- 🔔 Push notifications for safety alerts

## Prerequisites

- Node.js 16+
- Expo CLI (`npm install -g expo-cli`)
- iOS Simulator (Xcode) or Android Emulator (Android Studio)
- Physical device for testing camera/audio features
